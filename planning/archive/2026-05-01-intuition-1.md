# Intuition Phase — Initial Findings (5-bot dataset)

*2026-05-01 — Based on 4 notebooks analyzing 10 battles, 5 bots, 3 rounds each*

## Dataset

- **266,890 tick rows**, 22 predictor features, 3 CSV types (ticks, waves, scores)
- 5 robots: BeepBoop 2.0, Diamond 1.8.22, DrussGT 3.1.7, Saguaro 0.1, ScalarR 0.005h.053-noshield
- Missing data is systematic: opponent features are NaN when radar doesn't scan (not corruption)
- ~3.2% of scan ticks are firing events (heavily imbalanced)

## Notebook 1 — Data Overview

- Feature distributions are mostly bell-shaped, some right-skewed
- Each robot has a distinct behavioral fingerprint visible in box plots (different preferred distances, velocities, turning rates)
- Time-series plots show recognizable patterns: energy draining, distance oscillating, gun heat cycling
- Waves data shows robots use different bullet powers and fire at different distances

## Notebook 2 — Correlations

- Most features are weakly correlated (|r| < 0.3) — low redundancy, good for ML
- Strongest pair: `opponent_lateral_velocity ↔ opponent_heading_delta` (physically obvious: turning = lateral movement)
- `opponent_inferred_gun_heat` is the strongest single predictor of `opponent_fired` via point-biserial correlation
- **Feature relationships vary per robot** — Diamond's correlation structure differs from DrussGT's, confirming different strategies produce different statistical signatures

## Notebook 3 — Clustering

**PCA:**
- PC1 (~20% variance) loads on distance/energy; PC2 (~15%) loads on angles/lateral velocity
- Need ~18 components for 95% variance — data is genuinely high-dimensional
- Robots occupy slightly different regions in PCA space

**K-Means (K=3) found 3 natural game states:**

| Cluster | Profile |
|---------|---------|
| 0 | Long-range standoff — high distance, low velocity |
| 1 | Mid-range maneuvering — moderate distance, variable velocity |
| 2 | Close-range dogfight — low distance, high velocity |

- Clusters are roughly balanced (no single state dominates)
- DBSCAN confirmed 3-4 density-based clusters with ~2-5% noise/outliers
- Bullet patterns also cluster into 3 groups by power/distance/rate

## Notebook 4 — Simple ML

### Fire Prediction (main result)

| Model | Train Acc | Test Acc |
|-------|-----------|----------|
| Decision Tree (depth=5) | 0.9804 | 0.9769 |
| Decision Tree (unlimited) | 0.9998 | 0.9888 |
| **Random Forest (100 trees)** | **0.9915** | **0.9888** |
| Logistic Regression | 0.9885 | 0.9888 |

- Best: Random Forest — 98.88% accuracy, 97% recall, 95% precision
- Confusion matrix: ~370 true positives, ~20 false positives, ~11 false negatives out of 12,361 test ticks

### Top 5 Feature Importances

| Rank | Feature | Importance |
|------|---------|------------|
| 1 | **opponent_inferred_gun_heat** | **0.8845** |
| 2 | opponent_energy | 0.0326 |
| 3 | our_gun_heat | 0.0203 |
| 4 | distance_norm | 0.0100 |
| 5 | energy_ratio | 0.0089 |

- Gun heat is **27× more important** than the next feature — the model learns "gun is cool → fire imminent"
- Learning curves plateau early — more data provides marginal improvement for this task

### Round Winner Prediction

- 100 rounds, ~48% win rate (balanced)
- Random Forest: 70-73% accuracy (baseline 50%)
- Too small a dataset for reliable conclusions

## Key Takeaways

1. **Fire prediction is essentially solved by physics** — gun heat is a deterministic cooldown timer. Not an interesting ML problem.
2. **The real ML challenges are untested:**
   - Opponent movement prediction (where will they be when bullet arrives?)
   - Opponent aim prediction (what GuessFactor are they targeting?)
   - Strategic adaptation (how does behavior change over rounds?)
3. **Robot behavioral differences are real and measurable** — clustering and per-robot correlation differences confirm distinct strategies exist in the data.
4. **3 natural game states exist** — close/mid/long range combat. Models may benefit from state-aware predictions.
5. **Dataset was too small** — 5 bots, 3 rounds each. Need 50-bot, 35-round data for meaningful movement/targeting analysis.
